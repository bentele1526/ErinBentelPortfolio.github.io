Final Project Content

In my final project, I chose to use Python to create Horse Dodge. In this game I created multiple levels in which a horse character is used to jump over fences that fly across the screen. In order to continue to the next round, the horse character has to collect two apples. In the beginning, the user is asked to name the horse and during the game the horse changes costumes with the different arrow keys. I used lists to help me with the animation of the horse and classes for the different objects. There is a score to keep track of the points as well as a timer. I combined my passion with horses into a python game to create Horse Dodge.
